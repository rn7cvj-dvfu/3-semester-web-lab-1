3 семестр лабораторная работа веб № 1

___

To run:

- Install python
- Install requirements: `pip install -r requirements.txt`
- Run `python app.py` or ./tools/run.ps1

To build:

- Install python
- Install requirements: `pip install -r requirements.txt`
- Run `pyinstaller run.py` or ./tools/build.ps1
